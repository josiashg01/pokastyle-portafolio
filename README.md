# PokaStyle — Portafolio

Sitio web portafolio de **Poka (Josias Hidalgo Gil)** — Filmmaker y editor audiovisual en Santo Domingo, República Dominicana.

🌐 [pokastyle.lat](https://pokastyle.lat/)

## Stack

Sitio estático de una sola página: HTML + CSS + JavaScript vanilla, sin dependencias ni build.

## Despliegue

Desplegado en Vercel. Cualquier `push` a la rama `main` actualiza el sitio automáticamente.

## Estructura

- `index.html` — página completa (estilos y scripts embebidos)
- `stills/` — fotogramas y portadas de los proyectos
- `Fotos slide/` — foto de perfil
- `favicon.png`, `apple-touch-icon.png`, `og-image.jpg` — íconos e imagen para compartir
- `poka logo blanco.png` — logo
